## BLOB
Mapping data SQL BLOB pada Java. Blob adalah Binary Large Object (binary executable, gambar, video, atau objek multimedia lainnya). BLOB ini merupakan pointer yang menunjuk lokasi SQL BLOB, bukan isi datanya.
